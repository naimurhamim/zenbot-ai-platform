# ZenBot Nexus v2

Serverless AI Knowledge Platform built with AWS.

## Technologies

- AWS Lambda
- API Gateway
- DynamoDB
- Amazon S3
- PyPDF2
- JavaScript

## Features

- Chatbot interface
- Wikipedia knowledge search
- PDF question answering
- Session based chat history
- Login history
- Profile settings

## Architecture

Frontend → API Gateway → Lambda → DynamoDB
