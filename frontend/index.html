About This Project

ZenBot AI - v2 is a serverless chatbot project.

Built by MD Naimur Rashid, Sourav Chakraborty, and MD Rohan.

This system uses S3, API Gateway, AWS Lambda, DynamoDB, and Cognito-based setup resources.

It supports login-style access, chat history, session reuse, cloud-hosted backend processing, Wikipedia-based knowledge retrieval, and PDF-based educational Q&A.
